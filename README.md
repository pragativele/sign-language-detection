# sign-language-detection

Real-time Sign Language Detection using TensorFlow, OpenCV and MediaPipe

The system detects hand gestures and converts them into text and voice output.

## Technologies Used

- Python
- TensorFlow
- OpenCV
- MediaPipe

## Features

- Real-time gesture detection
- Hand landmark tracking
- Text generation
- Voice output

## Future Improvements

- More gesture support
- Mobile application
- Better accuracy
